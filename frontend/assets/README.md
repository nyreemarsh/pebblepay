# Frontend Assets

This directory contains all static assets for the PebblePay frontend application.

## Directory Structure

```
assets/
├── fonts/          # Custom font files
│   ├── .gitkeep
│   └── (font files: .ttf, .otf, .woff, .woff2)
│
└── images/          # Image files
    ├── .gitkeep
    ├── icons/      # Icon files (optional subdirectory)
    ├── logos/      # Logo files (optional subdirectory)
    ├── backgrounds/ # Background images (optional subdirectory)
    └── illustrations/ # Illustration files (optional subdirectory)
```

## Usage

### Fonts
Place custom font files in the `fonts/` directory. For a neon/cyberpunk aesthetic, consider:
- Futuristic/tech fonts
- Neon-style display fonts
- Monospace fonts for code blocks

### Images
Place image files in the `images/` directory. Supported formats:
- `.png` - For images with transparency
- `.jpg` / `.jpeg` - For photographs
- `.svg` - For scalable vector graphics (recommended for icons)
- `.webp` - For optimized web images
- `.gif` - For animations

## Recommended Fonts for Neon Aesthetic

- **Orbitron** - Futuristic, geometric
- **Rajdhani** - Modern, tech-inspired
- **Exo 2** - Clean, futuristic
- **Audiowide** - Bold, cyberpunk
- **Share Tech Mono** - Monospace, tech feel

## Notes

- All assets should be optimized for web use
- Use `.gitkeep` files to ensure empty directories are tracked by git
- Consider using a build tool to optimize and bundle assets
